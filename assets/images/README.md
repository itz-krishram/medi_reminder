# Image Assets

## Required Files

### empty_state.png
- **Purpose**: Display image when there are no medicines scheduled
- **Format**: PNG (with transparency recommended)
- **Recommended Size**: 256x256 to 512x512 pixels
- **Style**: Simple, friendly illustration related to medicine/health

## Where to Get Free Images

1. **Undraw.co**: https://undraw.co/search (search for "medicine" or "health")
2. **Icons8**: https://icons8.com/illustrations (free with attribution)
3. **Flaticon**: https://www.flaticon.com/ (free icons)

## Instructions

1. Download or create an empty state illustration
2. Save it as `empty_state.png`
3. Place it in this directory: `assets/images/empty_state.png`
4. Run `flutter clean` and rebuild the app

## Temporary Workaround

The app currently uses an icon-based empty state, so this image is optional.
The app will work fine without it until you add a custom illustration.

## Tips

- Use a light, friendly color scheme
- Medicine-related imagery works best (pills, bottles, etc.)
- Keep it simple and not too detailed
- Ensure good contrast with white/light backgrounds
