# Sound Assets

## Required Files

### alarm.mp3
- **Purpose**: Alarm sound that plays when medicine reminder triggers
- **Format**: MP3
- **Duration**: Recommended 10-30 seconds (will loop)
- **Volume**: Should be clearly audible but not jarring

## Where to Get Free Alarm Sounds

1. **Freesound.org**: https://freesound.org/browse/tags/alarm/
2. **Zapsplat**: https://www.zapsplat.com/sound-effect-category/alarms-and-sirens/
3. **Pixabay**: https://pixabay.com/sound-effects/search/alarm/

## Instructions

1. Download a suitable alarm sound (MP3 format)
2. Rename it to `alarm.mp3`
3. Place it in this directory: `assets/sounds/alarm.mp3`
4. Run `flutter clean` and rebuild the app

## Temporary Workaround

If you don't have an alarm sound yet:
- The app will still work but the alarm will use system default sound
- You can add the sound file later and rebuild
